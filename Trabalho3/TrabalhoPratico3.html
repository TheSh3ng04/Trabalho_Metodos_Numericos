(a) Pontos de Interpolao

Pretendese construir um conjunto de $n+1 = 8$ pontos $(x_i,f(x_i))_{i=0}^{7}$, de abcissas igualmente espaadas, que definem uma partio de $[-1,1]$ em $n=7$ subintervalos de igual amplitude.

A amplitude de cada subintervalo dada por $$ h = \frac{b-a}{n} = \frac{1-(-1)}{7} = \frac{2}{7}, $$ pelo que as abcissas so $ x_i = -1 + i\,h,\quad i=0,1,\dots,7. $

Utilizando esta expresso obtmse as abcissas $$ x_0=-1,\; x_1\approx -0{,}714286,\; x_2\approx -0{,}428571,\; x_3\approx -0{,}142857,\; x_4\approx 0{,}142857,\; x_5\approx 0{,}428571,\; x_6\approx 0{,}714286,\; x_7=1, $$ e, calculando $f(x_i)=(x_i)^2 + \sin(6x_i)$ com aritmtica de mquina, obtmse aproximadamente os seguintes valores:

Notese que $f$ a soma de um polinmio e de uma funo trigonomtrica suave, logo $f\in C^\infty([-1,1])$, o que garante que as condies de regularidade exigidas pelos teoremas de erro de interpolao polinomial e de splines cbicos naturais esto satisfeitas para este problema.

(b) -polinmio Interpolador e Spline Cbico Natural

Com o conjunto de 8 pontos equidistantes $(x_i, f(x_i))_{i=0}^{7}$ obtido na alnea (a), construmos primeiro o polinmio interpolador $p_7(x)$ recorrendo ao mtodo de Newton em diferenas divididas. Para isso, organizamos a tabela de diferenas divididas e calculamos os coeficientes

$$ c_0 = f[x_0],\; c_1 = f[x_0,x_1],\; \dots,\; c_7 = f[x_0,\dots,x_7], $$

que permitem escrever o polinmio interpolador na forma de Newton

$$ p_7(x)=c_0+c_1(x-x_0)+c_2(x-x_0)(x-x_1)+\dots+c_7(x-x_0)\cdots(x-x_6). $$

Para alm do polinmio interpolador global, determinamos tambm o spline cbico natural $s(x)$, que uma funo definida por partes, onde em cada subintervalo $[x_i, x_{i+1}]$ se tem um polinmio cbico, com as condies de continuidade

$$ s_i(x_{i+1}) = s_{i+1}(x_{i+1}), \quad s_i'(x_{i+1}) = s_{i+1}'(x_{i+1}), \quad s_i''(x_{i+1}) = s_{i+1}''(x_{i+1}), $$

e as condies de fronteira naturais

$$ s''(x_0) = s''(x_7) = 0. $$

n = 7                  # 8 pontos -> grau <= 7

# --- Polinmio de Newton p7(x), s com numpy/math ---

m = len(x_vals)
DD = [[0.0 for _ in range(m)] for _ in range(m)]
for i in range(m): DD[i][0] = y_vals[i]
for j in range(1, m):

        DD[i][j] = (DD[i+1][j-1] - DD[i][j-1]) / (x_vals[i+j] - x_vals[i])

coeffs = [DD[0][j] for j in range(m)]  # c0,...,c7

def p7(x):
    """Avalia o polinmio de Newton de grau <=7 em x."""
    value = 0.0
    term = 1.0
    for j in range(m):
        value += coeffs[j] * term
        term *= (x - x_vals[j])
    return value

# --- Spline cbico natural s(x) ---

h = x_vals[1] - x_vals[0]
A = np.zeros((n+1, n+1))
b_vec = np.zeros(n+1)

# Condies naturais
A[0, 0] = A[n, n] = 1.0

for i in range(1, n):
    A[i, i-1] = A[i, i+1] =h
    A[i, i]   = 4*h
    b_vec[i]  = 6 * (y_vals[i+1] - 2*y_vals[i] + y_vals[i-1]) / h

M = np.linalg.solve(A, b_vec)       #M_i = s''(x_i)

print("Valores de M_i = s''(x_i) para o spline cbico natural:\n")
for i in range(n+1): print(f"M_{i} = {M[i]: .6f}")
print("\n" + "-"*60 + "\n")

coefs_spline = []
for i in range(n):
    hi = x_vals[i+1] - x_vals[i]
    a_i = y_vals[i]
    c_i = M[i] / 2.0
    d_i = (M[i+1] - M[i]) / (6.0 * hi)
    b_i = (y_vals[i+1] - y_vals[i]) / hi - (2*M[i] + M[i+1]) * hi / 6.0
    coefs_spline.append((a_i, b_i, c_i, d_i))

print("Coeficientes dos polinmios cbicos S_i(x) em cada intervalo [x_i, x_{i+1}]:\n")

          f" + {b_i: .6f}(x - x_{i})"

def spline_natural(x):
    if x <= x_vals[0]: i = 0
    elif x >= x_vals[-1]: i = n-1
    else: i = np.searchsorted(x_vals, x) - 1
    a_i, b_i, c_i, d_i = coefs_spline[i]
    dx = x - x_vals[i]
    return a_i + b_i*dx + c_i*dx**2 + d_i*dx**3

No nosso programa de Python, aps construir a tabela de diferenas divididas, guardaramse os coeficientes de Newton $(c_0,\dots,c_7)$ e definiuse uma funo em Python que, dado um $x$, avalia diretamente o polinmio interpolador $p_7(x)$ usando a expresso anterior. Desta forma, o polinmio determinado apenas a partir dos coeficientes numricos calculados pelas diferenas divididas.

De seguida, utilizando o mesmo conjunto de pontos, construmos o spline cbico natural $s$. Considerouse um spline $s$ tal que, em cada subintervalo $[x_i,x_{i+1}]$, coincide com um polinmio cbico $S_i(x)$, com continuidade da funo e das suas derivadas primeira e segunda nos ns interiores, e impondo ainda as condies naturais $s''(x_0)=s''(x_7)=0$. Para isso, montouse o sistema tridiagonal clssico para os valores $M_i = s''(x_i)$, $(i=0,\dots,7)$, que no caso de malha uniforme $h=x_{i+1}-x_i$ assume, para $(i=1,\dots,6)$, a forma $$ h\,M_{i-1} + 4h\,M_i + h\,M_{i+1} = 6\left(\frac{f(x_{i+1})-f(x_i)}{h} - \frac{f(x_i)-f(x_{i-1})}{h}\right), $$
com $M_0 = M_7 = 0$. O sistema linear foi resolvido com numpy.linalg.solve e, a partir dos $(M_i)$, determinaramse os coeficientes $(a_i,b_i,c_i,d_i)$ dos polinmios cbicos[1][2] $$ S_i(x)=a_i+b_i(x-x_i)+c_i(x-x_i)^2+d_i(x-x_i)^3,\quad i=0,\dots,6, $$
definindo assim o spline cbico natural $s(x)$ em todo o intervalo $[-1,1]$.

Deste modo, obtemos duas aproximaes interpolantes da funo $(f(x)=x^2 + \sin(6x))$: o polinmio interpolador global $(p_7(x))$, construdo pelo mtodo de Newton em diferenas divididas, e o spline cbico natural $s(x)$, definido seccionalmente, que sero comparados graficamente com a funo original e atravs das funes erro nas alneas seguintes.

Comparao entre o polinmio interpolador e o spline cbico natural

1. Polinmio interpolador $p_7(x)$

O polinmio interpolador de grau 7 ajusta exatamente os 8 pontos de interpolao.
No entanto, como um polinmio global de grau elevado, tende a apresentar oscilaes significativas entre os pontos, especialmente perto dos extremos do intervalo $[-1,1]$.
Este fenmeno conhecido como efeito de Runge e ocorre frequentemente quando se usam pontos uniformemente espaados.

Como consequncia:

  • O erro $|f - p|$ aumenta consideravelmente nos extremos.
  • A aproximao menos estvel e menos fiel ao comportamento local da funo.

2. Spline cbico natural $s(x)$

O spline cbico natural construdo como uma coleo de polinmios cbicos em cada subintervalo, garantindo continuidade das derivadas primeira e segunda.
Este mtodo evita oscilaes indesejadas e adapta-se muito melhor ao comportamento local da funo.

Vantagens observadas:

  • A aproximao mais suave e localmente precisa.
  • O erro $|f - s|$ muito mais pequeno e distribudo de forma uniforme.
  • O spline no sofre do efeito de Runge, mesmo usando pontos equidistantes.

3. Comparao grfica dos erros

A anlise dos grficos do erro absoluto mostra que:

  • O erro do polinmio interpolador $|f - p|$ apresenta picos elevados, sobretudo junto aos extremos.
  • O erro do spline cbico natural $|f - s|$ reduzido e mantm-se consistentemente baixo ao longo de todo o intervalo.

Concluso

O spline cbico natural fornece uma aproximao muito mais estvel e precisa da funo do que o polinmio interpolador global de grau elevado. Enquanto o polinmio tenta ajustar todos os pontos com um nico polinmio de grau elevadotornando-se oscilatrioo spline utiliza aproximaes locais suaves, resultando numa interpolao mais fiel e com menor erro.

(c) Majorantes dos erros em = 0.1 e = 0.9

Pretendese calcular majorantes dos erros cometidos ao estimar os valores de $f(0.1)$ e $f(0.9)$ usando o polinmio interpolador $p_7$ e o spline cbico natural $s$.

Para o polinmio interpolador, aplicamos o teorema do erro na interpolao polinomial apresentado nas aulas tericas. Como $f\in C^{8}([-1,1])$, existe, para cada $x\in[-1,1]$, um ponto $c(x)$ tal que $$ f(x) - p_7(x) = \frac{f^{(8)}(c(x))}{8!}\,(x-x_0)\cdots(x-x_7). $$ Daqui resulta a estimativa $$ |f(x) - p_7(x)| \leq \frac{\max_{t\in[-1,1]}|f^{(8)}(t)|}{8!}\,\prod_{i=0}^{7}|x-x_i|. $$ onde o fator dominante no comportamento do erro precisamente o produto $\prod_{i=0}^{7}(x-x_i)$.

No caso do spline cbico natural, usase a forma geral do erro para splines cbicos naturais, que envolve a quarta derivada de $f$. Para $x\in[x_i,x_{i+1}]$, temse $$ f(x) - s(x) = \frac{5f^{(4)}(\eta(x))h^4}{384}, $$ para algum $(\eta(x)\in[x_i,x_{i+1}])$, onde $h = \max_{i=1,2,...,n}\bigl[h_i]$ e $h_i = x_i - x_{i-1}$ donde se obtm a majorante

$$ |f(x) - s(x)| \leq \frac{5\max_{t\in[-1,1]}|f^{(4)}(t)|h^4}{384}\ $$

Temos que: $ f^{(4)} = 6^4sin(6x) = 1296\sin(6x) $ e $ f^{(8)} = 6^8\sin(6x) = 1679616\sin(6x) $

De seguida, recorrese avaliao direta de $(f)$, $(p_7)$ e $(s)$, para obter majorantes e erros efetivos em $(x=0{,}1)$ e $(x=0{,}9)$.

Considerando o conjunto de dados correspondente s mdias mensais da evaporao da gua (em polegadas) em Adelaide, ao longo de 23 anos, fornecido no enunciado sob a forma de 12 pares ((x_i,f(x_i))), onde (x_i) representa o ms (de 1 a 12) e (f(x_i)) o valor mdio de evaporao nesse ms. Assim, trabalhouse com os pontos

$$ x_i = 1,2,\dots,12,\quad f(x_i) \in \{8{,}6,\;7{,}0,\;6{,}4,\;4{,}0,\;2{,}8,\;1{,}8,\;1{,}8,\;2{,}1,\;3{,}2,\;4{,}7,\;6{,}2,\;7{,}6\}. $$

Este conjunto de 12 pontos define uma funo discreta que se pretende aproximar por um polinmio interpolador global e por um spline cbico natural, de forma a estudar o comportamento sazonal da evaporao ao longo do ano.

Com base nesses dados, construmos o polinmio interpolador $p(x)$ de grau 11 recorrendo ao mtodo de Newton em diferenas divididas. Para isso, geramos a tabela de diferenas divididas $\,f[x_i,\dots,x_j]\,$ e extramos os coeficientes $$ c_0=f[x_1],\;c_1=f[x_1,x_2],\;\dots,\;c_{11}=f[x_1,\dots,x_{12}], $$
que permitem escrever o polinmio na forma de Newton, $$ p(x)=c_0+c_1(x-x_1)+c_2(x-x_1)(x-x_2)+\dots+c_{11}(x-x_1)\cdots(x-x_{11}). $$

De seguida, utilizando o mesmo conjunto de pontos, construmos o spline cbico natural $(s)$. Considerabdo um spline $s$ tal que, em cada subintervalo $[x_i,x_{i+1}]$, coincide com um polinmio cbico $S_i(x)$, com continuidade da funo e das suas derivadas primeira e segunda nos ns interiores, e impondo ainda as condies naturais $s''(x_1)=s''(x_{12})=0$. Para isso, montouse o sistema tridiagonal clssico para os valores $M_i = s"(x_i)$, $i=1,\dots,12$, o qual foi resolvido numericamente, e a partir dos $M_i$ determinaramse os coeficientes $a_i,b_i,c_i,d_i$ dos polinmios cbicos que definem $s(x)$ em cada intervalo $[x_i,x_{i+1}]$.

Por fim, representamos graficamente, no intervalo $x\in[1,12]$, os dados experimentais, a aproximao polinomial $p(x)$ e o spline cbico natural $s(x)$, de modo a comparar visualmente o comportamento das duas aproximaes da funo de evaporao ao longo do ano.

# Dados do enunciado (meses e evaporao) [file:2]
x_mes = np.arange(1, 13, dtype=float)  # 1,...,12
y_evap = np.array([8.6, 7.0, 6.4, 4.0, 2.8, 1.8, 1.8, 2.1, 3.2, 4.7, 6.2, 7.6])

# --- Polinmio interpolador p(x) por Newton (diferenas divididas) ---

m = len(x_mes)
DD = [[0.0 for _ in range(m)] for _ in range(m)]
for i in range(m): DD[i][0] = y_evap[i]
for j in range(1, m):
    for i in range(m - j):
        DD[i][j] = (DD[i+1][j-1] - DD[i][j-1]) / (x_mes[i+j] - x_mes[i])

coeffs_p = [DD[0][j] for j in range(m)]  # c0,...,c11

def p_interp(x):
    """Avalia o polinmio interpolador p(x) de grau <=11 em x."""
    value = 0.0
    term = 1.0
    for j in range(m):
        value += coeffs_p[j] * term
        term *= (x - x_mes[j])
    return value

# --- Spline cbico natural s(x) nos 12 pontos ---

n = m - 1                    # 11 intervalos
h = np.diff(x_mes)           # aqui h_i = 1 para todos, mas escrevemos em geral

A = np.zeros((m, m))
b_vec = np.zeros(m)

# Condies naturais: M_1 = 0, M_12 = 0
A[0, 0] = 1.0
A[m-1, m-1] = 1.0
b_vec[0] = 0.0
b_vec[m-1] = 0.0

for i in range(1, m-1):
    hi = h[i-1]
    hi1 = h[i]
    A[i, i-1] = hi
    A[i, i]   = 2*(hi + hi1)
    A[i, i+1] = hi1
    b_vec[i]  = 6 * ((y_evap[i+1] - y_evap[i]) / hi1 - (y_evap[i] - y_evap[i-1]) / hi)

M = np.linalg.solve(A, b_vec)   # M[i] = s''(x_i)

# Coeficientes dos cbicos S_i(x) em cada [x_i, x_{i+1}]
coefs_s = []
for i in range(n):
    hi = x_mes[i+1] - x_mes[i]
    a_i = y_evap[i]
    c_i = M[i] / 2.0
    d_i = (M[i+1] - M[i]) / (6.0 * hi)
    b_i = (y_evap[i+1] - y_evap[i]) / hi - (2*M[i] + M[i+1]) * hi / 6.0
    coefs_s.append((a_i, b_i, c_i, d_i))

def s_spline(x):
    """Avalia o spline cbico natural s(x) nos dados de evaporao."""
    if x <= x_mes[0]: i = 0
    elif x >= x_mes[-1]: i = n-1
    else: i = np.searchsorted(x_mes, x) - 1
    a_i, b_i, c_i, d_i = coefs_s[i]
    dx = x - x_mes[i]
    return a_i + b_i*dx + c_i*dx**2 + d_i*dx**3

Em seguida, utilizando o mesmo conjunto de 12 pontos, construmos o spline cbico natural $s(x)$. Considerouse um spline cbico tal que, em cada subintervalo $[x_i,x_{i+1}]$, coincide com um polinmio cbico $$ S_i(x)=a_i+b_i(x-x_i)+c_i(x-x_i)^2+d_i(x-x_i)^3,\quad i=1,\dots,11, $$ com continuidade de $(s)$, $(s')$ e $(s'')$ nos ns interiores e impondo ainda as condies naturais $(s''(x_1)=s''(x_{12})=0)$.
